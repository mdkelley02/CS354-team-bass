Translators and Installation Instructions

To be able to compile Rust source code and develop programs in Rust you need two things.

  • The Rust Compiler
  • A Linker

The Rust compiler will generate machine code from the source programs and the linker will join the compiled output into a single file.

Follow the instructions below for your specific platform to begin using Rust.

Windows

Download and install the Rust compiler here and install C++ Build Tools from Visual Studio here.

Mac or Linux

Download the Rust compiler by executing the following command. This will download rustup onto your machine.


        $ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh

If you are unsure whether or not you have a linker you can try running a rust program and see if you get linker errors. In the case of linker errors you need to install a C compiler.

On Mac you can do this with the following command.


        $ xcode-select --install

On Linux you can do this with the following commands.


        $ sudo apt-get update
        $ sudo apt install build-essential

You can find direct instructions for installing Rust from the Rust documentation here.
