Introductory Programs

Build and Run Instructions (For All Programs)

  1. Ensure all dependencies for your platform are installed by following these instructions here.
  2. Copy one of the desired programs below into a file with a .rs extension. For example, linked_list.rs.
  3. From the command line, ensure you are in the directory of the copied program, then run the following commands:

(Windows)


        $ rustc your_program_name.rs
        $ your_program_name.exe

(Mac & Linux)


        $ rustc your_program_name.rs
        $ ./your_program_name

Basic Struct and Enum

(Author: Taylor Brooks)

Running this program will output the following:

Student { name: "Joe", grade: B, age: 20, major: "Mechanical Engineering" }
Student { name: "Joe", grade: A, age: 20, major: "Mechanical Engineering" }


        #[derive(Debug)]
        enum Grade {
            A,
            B,
            C,
            D,
            F,
        }

        #[derive(Debug)]
        struct Student {
            name: &'static str,
            grade: Grade,
            age: i32,
            major: String
        }

        impl Student {
            fn changeGrade(&mut self, newGrade: Grade)  {
                self.grade = newGrade;
            }
        }

        fn main() {
            let mut student = Student {name: "Joe", grade: Grade::B, age: 20, major: "Mechanical Engineering".to_string()  };
            println!("{:?}", student);
            student.changeGrade(Grade::A);
            println!("{:?}", student);
        }

Basic Lambdas and Higher Order Functions

(Author: Cory Tomlinson)

Running this program will output the following:

5050
3628800


        use std::ops::{Mul, RangeInclusive};

        fn sum_range(range:RangeInclusive<i32>)->i32 {
           range.fold(0,|acc,n| acc+n)
        }

        fn op_range<F>(range:RangeInclusive<i32>,op:F)->Option<i32> where F:Fn(i32,i32)->i32 {
           range.reduce(|acc,n| op(acc,n))
        }

        fn main() {
           println!("{}",sum_range(1..=100));
           println!("{}",match op_range(1..=10,i32::mul) {
              Some(result) => result,
              None => panic!("op_range returned an empty value")
           });
        }
